## Commander's Intent

- **Purpose:** Seven verified fidelity gaps in the `/war` engine's audit/gate/dispatch spine share one shape — an engine-side record, routing set, or prompt clause diverges from what the dispatched agent is told or reports back. None corrupts a run, but each silently weakens bookkeeping or evidence: subset baseline-debt reports mint duplicate debt entries (#798), pin-mismatch-demoted findings can still ride `--ace` (#805), a `requiresTest:false` task's files leak into its successor's `--mapped` evidence set (#806), the `COST_CLAIM_SHARED` byte anchor is a quote-lint landmine with no warning (#811), the land-site gate prompt states a wrong cwd (#815), the floor-retry fix dispatch ignores the operator's `agents.worker.fix` tier (#817), and the land-authoritative integrated-tip seat is structurally locked out of its own HARD path because `gate_log_path` never reaches it (#818, the one major). Close all seven with behavior-true records and prompt-true prose.
- **Method:** Single-commit engine+test+refiner-prompt task (every mechanic shares `workflow-template.js`/`workflow-template.test.mjs`, and the both-surfaces rule chains `agents/war-refiner.md` in for #818), plus a file-disjoint doc task for the #815 learning resolution note, plus a trailing release phase. Guardrails: no enum surface moves (`land-decision.mjs`, its hand-mirrored copy, and the drift-guard test are untouched — ADR 0005); every fail-open boundary stays fail-open (absent `gate_log_path` ⇒ SOFT, absent fix-tier config ⇒ base tier, empty debt ⇒ byte-identical prompts, sentinel `integration_sha` ⇒ tracker unchanged); demotions never drop silently (ADR 0013 — `originalSeverity` + absence note preserved); prompt-substring locks are updated in the same commit, never loosened to pass. #817 deliberately closes the gap the war-room-config-expansion Task 1.2 audit recorded as intentional — that is scope, not creep.
